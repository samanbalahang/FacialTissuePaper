# Facial Tissue Paper WordPress Theme

A modern and responsive WordPress theme designed for facial tissue paper, hygiene products, and related businesses. Built with Tailwind CSS and optimized for performance, clean design, and easy customization.

## Demo

Live Preview:

🚀 [View Demo](http://samanbalahang.github.io/FacialTissuePaper/index.html)

## Features

- Responsive design
- Built with Tailwind CSS
- Lightweight and optimized assets
- Modern UI/UX
- Easy WordPress integration
- Cross-browser compatibility

## Requirements

Before working with the theme source files, make sure the following software is installed:

- Node.js (Latest LTS recommended)
- npm (Included with Node.js)

Download Node.js from:

https://nodejs.org/

## Installation

### 1. Install the Theme

Upload the theme folder (`bala`) to:

```text
wp-content/themes/
```

or install it through the WordPress Admin Dashboard.

### 2. Activate the Theme

Go to:

```text
WordPress Dashboard → Appearance → Themes
```

Activate **Bala**.

## Development Setup

Navigate to the theme directory:

```bash
cd wp-content/themes/bala
```

Install dependencies:

```bash
npm install
```

## Building CSS

The theme uses Tailwind CSS as the source stylesheet.

Source file:

```text
bala/tailwind/style.css
```

Compiled output:

```text
bala/assets/css/style.min.css
```

After modifying Tailwind classes or styles, rebuild the CSS using the project's npm scripts.

Example:

```bash
npm run build
```

or

```bash
npm run watch
```

*(Use the scripts defined in your package.json.)*

## Customization

### Tailwind Styles

Edit:

```text
bala/tailwind/style.css
```

Then rebuild the CSS.

### Compiled CSS

Generated file:

```text
bala/assets/css/style.min.css
```

Do not edit this file directly, as it will be overwritten during compilation.
